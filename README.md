# tvOS-26.1-Jailbreak
This is a jailbreak for tvOS 26.1 that uses palera1n

Thanks to sawyerthemiller for providing the guide on how you do this.

There is a whole missing folder from the source code because their is too many files for github to handle at once.

This is tested on an Apple TV HD 4th Generation
# Usage
Download the correct version for your OS.
> [!NOTE]
>You need to factory reset your apple TV before starting
